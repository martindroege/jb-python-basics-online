Dateien lesen 1¶

f = open("DATEI.txt", "r", encoding="utf-8")
text = f.read()
f.close()

Dateien lesen 2¶

with open("DATEI.txt", "r", encoding="utf-8") as f:
    text = f.read()

I/O Parameter¶

  • “r” = read / lesen

  • “w” = write / schreiben

  • “a” = append / anhängen
